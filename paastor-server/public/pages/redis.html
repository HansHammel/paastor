Redis on Paastor

With a few clicks, you can have a Redis instance running on any of your servers.

From the Server Manager, hit the Redis checkbox for one of your servers and fill out the required fields. In a few moments you will have Redis installed.

Changing the password / lost password

If you lose the Redis password, want to change it, or want to remove it, log into the server as root and edit requirepass in /etc/redis/redis.conf.
