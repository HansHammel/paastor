SSL on Paastor

SSL is available on paid accounts only.
You don't have to pay very much, but you have to be paying something!

Add SSL certs and keys in the Server Manager.

SSL certificates can be re-uploaded an unlimited number of times from the web UI - in case it changes, or you make a mistake.

What happens on the server

Sheep contains a proxy and a router with support for SSL.

When you add SSL to an app, the proxy will use the certificates during the requests on behalf of your app, then route the traffic to your app on process.env.PAASTOR_PORT.

This setup is similar to nginx or other proxy servers.
